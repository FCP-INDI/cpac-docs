How to Use C-PAC

Ways to Run

  • Local machine with Docker and/or Singularity
    • The recommended way to run C-PAC is through the use of a container such as Docker or Singularity.
    • Using these containers can be facilitated by the cpac Python package.
  • HPC with Singularity
  • AWS
  • brainlife.io
  • Flywheel
  • Starcluster on AWS or HPC
