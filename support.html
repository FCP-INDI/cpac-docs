User Support

This is a beta version of C-PAC, which means that it is still under active development. As such, although we have done our best to ensure a stable pipeline, there will likely still be a few bugs that we did not catch.

For a general troubleshooting guide, please refer to the C-PAC Error Triage Flow. If you find a bug or would like to suggest a new feature, please open an issue on the the C-PAC Github issue tracker. For more general questions or comments about C-PAC, we invite you to contribute to the C-PAC Neurostars discussion forum.
